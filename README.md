#### *Set Port: http://localhost:8080*

*[ User ] variable*
    
    - id: DB에 저장되는 user 자체의 id (신경 안 써도 될듯)

    - language : 언어

    - country : 출신 국가

    - firstname : 이름

    - lastname : 성

    - month : 월 (생일)

    - day : 일 (생일)

    - year : 년 (생일)

    - loginId : 아이디

    - password : 비밀번호

    - confirmPassword : 비밀번호 확인

    - gender : 성별

    - phoneNumber : 전화번호

    - certification_number : 인증번호
